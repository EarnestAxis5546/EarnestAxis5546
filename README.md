## Project Spotlight

My passion is channeled into **[NovaSentinel](https://github.com/earnestaxis5546/NovaSentinel)**, a state-of-the-art Anti-DDoS system for server protection.

- **[NovaSentinel](https://github.com/earnestaxis5546/NovaSentinel)** 🛡️  
  A high-performance Anti-DDoS system built in C++20 on Arch Linux, designed to protect servers and cloud infrastructure from distributed denial-of-service attacks. NovaSentinel features real-time threat detection, multithreaded packet analysis, and advanced mitigation algorithms to ensure minimal latency and maximum uptime. Its lightweight, scalable architecture makes it ideal for enterprise servers, data centers, and critical infrastructure. Optimized with modern C++ standards and Arch Linux’s efficiency, it delivers robust security for high-traffic environments. (Active Development 🔥)
